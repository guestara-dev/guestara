# Onix 🏨

Hotel Management con IA — MVP Dashboard

## Setup

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## Estructura

```
onix/
├── app/
│   ├── layout.tsx
│   └── page.tsx
├── components/
│   ├── Sidebar.tsx
│   ├── StatsGrid.tsx
│   ├── RoomGrid.tsx
│   └── ReservationModal.tsx
├── lib/
│   └── data.ts
```

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- lucide-react (iconos)
